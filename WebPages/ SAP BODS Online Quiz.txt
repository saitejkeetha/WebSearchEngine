Q 1 - Which of the following component is responsible for extraction, transformation and load of data using Data Service designer? A - Work Flow B - Data Flow C - Replication job D - Error Flow Answer : B Explanation Data flows extract, transform, and load data. Everything having to do with data, including reading sources, transforming data, and loading targets, occurs inside a data flow.
