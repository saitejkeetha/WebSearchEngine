A - Enterprise J2EE Bean B - Enterprise Java Bean C - Enterprise Java oBject D - Entity Java oBject Answer : B Explaination EJB stands for Enterprise Java Bean. Show Answer Q 2 - Which of the following is true about EJB? A - EJB is an essential part of a J2EE platform. B - EJB provides an architecture to develop and deploy component based enterprise applications considering robustness, high scalability and high performance. C - EJB stands for Enterprise Java Bean. D - All of the above. Answer : D Explaination EJB stands for Enterprise Java Beans. EJB is an essential part of a J2EE platform. J2EE platform have component based architecture to provide multi-tiered, distributed and highly transactional features to enterprise level applications. EJB provides an architecture to develop and deploy component based enterprise applications considering robustness, high scalability and high performance. Answer : D Explaination EJB provides all of the above mentioned benefits. Show Answer Q 4 - Which of the following is true about session bean?
