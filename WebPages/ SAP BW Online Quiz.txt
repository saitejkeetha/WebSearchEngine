You can use Next Quiz button to check new set of questions in the quiz. Q 1 - To use special characters in BW, which of the following transaction can be used? A - RSA1 B - RSA15 C - RSKC D - RRBS Answer : C Explanation RSKC is used to enter special characters in BW. Show Answer Q 2 - Which of the following component in BW architecture can be used for detailed storage of data and used for BEx reporting? A - PSA B - Operational Data Store C - InfoCubes D - Infoarea Answer : B Explanation In SAP BW architecture, ODS can be used for detailed storage of data and used for BEx reporting. It exists between PSA and InfoCubes. Show Answer Q 3 - When you upload the same data again in an Operational data store in overwrite mode, it creates new entries in change log. A - True B - False Answer : B Show Answer Q 4 - To save in progress query so that it is not accessible to other users. It should be saved as? A - Public B - Network Drive C - Local System D - Favorites Answer : D Explanation You can store all the under development queries to Favorites to prevent them accessible to other users. Show Answer Q 5 - Which of the following are part of Dimension tables in BW? A - Pointers to text values B - SID's & Pointers to Fact table C - Transfer Rules D - None Answer : B Explanation Dimension tables in SAP BW contains SID and pointers to fact table. Show Answer Q 6 - In SAP BW, to create new InfoObjects to store data, which of the following tab is used in Administration workbench? A - Administration B - Data Source C - Modeling D - Translation Answer : C Explanation In RSA1, you use Modeling tab to create new objects to store BI data. Show Answer Q 7 - In a Log view under process change management, a yellow mark shows that a process chain A - Not Yet Run B - Completed with no errors C - Running D - Failed Answer : C Explanation Grey: Not Yet Run Green: Completed with no errors Yellow: Running Red: Failed Show Answer Q 8 - A company wants to set up BI reporting on Sales data from source system without extraction. Which of the following SAP BW feature support this? A - MultiProvider B - Delta Load C - Virtual Provider D - InfoSets Answer : C Explanation You can access BI data from source system without extraction using Virtual Providers. Show Answer Q 9 - Virtual Providers only allow read access to data in source system? A - True B - False Answer : A Show Answer Q 10 - Which of the following class contains majority of authorization objects in BI? A - AS B - DS C - RS D - ZS Answer : C Explanation Authorization objects are grouped according to authorization object classes and majority of objects are in class RS in BI.
